Notebook criado para exploração de dados de um dataset, obtidos através do Kaggle.

Na célula 5, podemos inserir como serão filtrados os dados:
 - country_ref: País
 - year_ref: Ano
 - item_ref: Coluna de dados (Nitrogen Oxide, Sulphur Dioxide, Carbon Monoxide, Organic Carbon, NMVOCs, Black Carbon, Ammonia)
 - num_ranking: Quantidade de países que serão incluídos nos gráficos
 
A partir da célula 12, estão os gráficos.
