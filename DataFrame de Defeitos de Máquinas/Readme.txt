Para treinar a questão de análise por gráficos de paradas geradas por manutenção corretiva, foi elaborado esse programa para gerar dados aleatórios.
Nesse programa consta:
1) Máquinas
2) Peças presentes nas máquinas
3) Defeitos possíveis para cada peça
4) Tempo de reparo
5) Data e hora dos chamados para manutenção
